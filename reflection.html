In this world of Web development and Web designing the major thing kept in mind while developing websites is its appearance to user and the comfortability that user gets while working on website or using the website and this whole design gets or turns to a lack hole when the same website is run on different sized screen i.e. mobile, tablet, desktop and laptops Which affects the efficiency while working on it and content display to the user and sometimes even whole website and its design gets distorted while testing it on another screen and This problem in web development is overcome by a special technique in web world called “Responsive web Designing (RWD)” and this technique is defined as Responsive web design (RWD) is a web development approach that creates dynamic changes to the appearance of a website, depending on the screen size and orientation of the device being used to view it and this technique in html is implemented with help of CSS.

RWD relates very closely with the web technologies and the relation for every web developing learner is important as if it is not developed the website may fail to run or may lose its appearance on different display sizes automatically without changing any background html code. So, today a developer no longer needs to write different websites for different sizes of display on which content is to be displayed and this also, makes other factors such as

  1. 1.Cost effectiveness

  2. 2. Flexibility

  3. 3. Improved user experience

  4. 4. Search engine optimization gains

  5. 5. Ease of management

And all of these above factors directly relate with web development and developers as they now days no longer need to update the web sites to different sizes and make them easy to manage by designer and easy to use for user.

After learning about responsive web designs and web designing, I learned its importance of being presence in each and every website being developed modernly as being in existence in field of web designing is very important as these websites are easy to maintain and more easy to make changes in updating website in future as we developers will no longer need to take care of web appearance on different devices after updating as its by default build with responsive design by default and moreover, its important to be and learn the latest version of everything so, I think every new learner should be well equipped with RWD use and its implementing.
